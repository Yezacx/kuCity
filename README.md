# kuCity城市选择插件
#### 在页面中引入input标签
   ```
   <input type="text" class="search" placeholder="选择城市">
   ```
#### 引入kuCity.css文件
   ```
   <link rel="stylesheet" href="./css/kuCity.css">
   ```
#### 先引入jQuery再引入kuCity.js文件
   ```
   <script src="./js/jquery-3.2.1.js"></script>
   <script src="./js/kuCity.js"></script>
   ```
#### 调用
   ```
   $(".search").kuCity();
   ```
